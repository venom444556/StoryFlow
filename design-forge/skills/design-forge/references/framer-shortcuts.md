# Framer Keyboard Shortcuts

Complete reference for CDP key automation. Use `forge key` for combos.

---

## Element Creation
| Shortcut | Action | forge command |
|----------|--------|---------------|
| F | Frame | `forge insert frame` |
| T | Text | `forge insert text` |
| S | Stack | `forge insert stack` |
| L | Link | `forge insert link` |
| I | Image | `forge insert image` |
| V | Video | `forge insert video` |
| C | Code component | `forge insert code` |

## Selection & Navigation
| Shortcut | Action | forge command |
|----------|--------|---------------|
| Cmd+A | Select all | `forge key "Meta+a"` |
| Escape | Deselect / exit mode | `forge key "Escape"` |
| Tab | Select next sibling | `forge key "Tab"` |
| Shift+Tab | Select previous sibling | `forge key "Shift+Tab"` |
| Enter | Select child | `forge key "Enter"` |
| Shift+Enter | Select parent | `forge key "Shift+Enter"` |

## Transform
| Shortcut | Action |
|----------|--------|
| Cmd+D | Duplicate |
| Cmd+G | Group |
| Cmd+Shift+G | Ungroup |
| Alt+drag | Clone drag |
| Arrow keys | Move 1px |
| Shift+Arrow | Move 10px |
| Cmd+[ | Send backward |
| Cmd+] | Bring forward |
| Cmd+Shift+[ | Send to back |
| Cmd+Shift+] | Bring to front |

## Editing
| Shortcut | Action |
|----------|--------|
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |
| Cmd+C | Copy |
| Cmd+V | Paste |
| Cmd+X | Cut |
| Backspace | Delete |
| Cmd+Shift+L | Lock/Unlock |
| Cmd+Shift+H | Show/Hide |
| 0-9 | Opacity (1=10%...0=100%) |

## View
| Shortcut | Action |
|----------|--------|
| Cmd+0 | Zoom to 100% |
| Cmd+1 | Zoom to fit |
| Cmd+2 | Zoom to selection |
| Cmd++ | Zoom in |
| Cmd+- | Zoom out |
| Space+drag | Pan canvas |

## Layout
| Shortcut | Action |
|----------|--------|
| Shift+A | Add auto-layout (stack) |
| Cmd+Shift+A | Remove auto-layout |

## Text
| Shortcut | Action |
|----------|--------|
| Cmd+B | Bold |
| Cmd+I | Italic |
| Cmd+U | Underline |
| Cmd+Shift+. | Increase font size |
| Cmd+Shift+, | Decrease font size |
